# 🌌 Dark_Weather

Aplicación web moderna para consultar el clima actual y el pronóstico extendido de cualquier ciudad del mundo.  
Diseñada con un **tema oscuro elegante**, autocompletado dinámico, historial de ciudades y **íconos personalizados** de Weather Icons.

---

## 🚀 Características

- 🌍 **Búsqueda de ciudades** con autocompletado dinámico.
- 📊 **Clima actual**: temperatura, humedad y descripción.
- ⏳ **Pronóstico extendido**: próximos intervalos con íconos personalizados.
- 🕑 **Historial de ciudades**: guarda las últimas búsquedas en `localStorage`.
- 🎨 **Diseño oscuro responsivo**: se adapta a móvil y escritorio.
- 🌤️ **Íconos vectoriales** con [Weather Icons](https://erikflowers.github.io/weather-icons/).

---

## 🛠️ Tecnologías utilizadas

- [Vite](https://vitejs.dev/) para el entorno de desarrollo.
- [Bootstrap 5](https://getbootstrap.com/) para estilos responsivos.
- [Weather Icons](https://erikflowers.github.io/weather-icons/) para íconos personalizados.
- [OpenWeather API](https://openweathermap.org/api) para datos meteorológicos.
